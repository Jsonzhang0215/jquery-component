jquery-component
================

#日常通用组件

###Depends:  
jquery-1.11.0  

##jquery-simple-slider
```javascript  
var config = {
    width: 670,
    height: 240
};
var slider =  new SimpleSlider($("#slider"),config);
```
###config  
>width: Number  
height: Number  
duration: Number (ms)  
autoPlay: Bool  
hasControl: Bool  
hasTurnPageBtn: Bool
